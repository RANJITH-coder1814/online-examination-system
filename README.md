# online-examination-system
A web-based Online Examination System where:  👨‍🎓 Students can register, login, and take exams  🧑‍💼 Admin can add, update, and delete questions  📊 System automatically calculates results  📝 Multiple-choice questions (MCQs)  This project demonstrates authentication, database integration, REST APIs, and exam logic handling.

A web-based Online Exam platform built using Node.js, Express, and MySQL.

## 🚀 Features
- Student Login
- Admin Question Management
- MCQ-based Exam
- Automatic Result Calculation
- Database Integration

## 🛠 Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express
Database: MySQL

## 📦 Setup Instructions

1. Create MySQL database:
   CREATE DATABASE exam_system;

2. Run the SQL table queries

3. Install dependencies:
   npm install express mysql2 cors

4. Run server:
   node server.js

Server runs at:
http://localhost:5000
